# Edutracker_Google_Sheets_extension
It helps turn a spreadsheet into an activity tracker and to-do list

It will sort the sheet based first on particular columns: first week number and second based on weekday. Thus, the user may type in a new activity whereever (e.g. at the bottom of the sheet) and it will be moved to the appropriate row for future use. Could not be achieved with formulas. 

Uses App Script feature in Google Workspace.
